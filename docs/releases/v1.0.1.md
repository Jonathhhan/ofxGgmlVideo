# ofxGgmlVideo v1.0.1

## Compatibility

- openFrameworks: `of_v20260406_vs_64_release`
- minimum `ofxGgmlCore`: `1.0.1` / commit `2171e8b`
- `OFXGGML_VIDEO_VERSION_STRING`: `1.0.1`

## Highlights

- Added independent Video addon version metadata.
- Exposed version metadata through `ofxGgmlVideo.h`.
- Documented the release checklist and release policy for video workflows.
- Kept video-specific request helpers out of `ofxGgmlCore`.
- Preserved the root-level `ofxGgmlVideoFrameExample` as the first example lane.

## Runtime Scope

This release includes public request/result helper types, input validation,
description helpers, a root-level video frame example skeleton, local
validation, and headless tests.

It does not include a complete video understanding, temporal analysis,
image-to-video, temporal diffusion, transformer sequence, or GAN video runtime
adapter yet. Those remain roadmap tasks.

- None relative to the new companion-addon line.
